# Hello World Project

## Description
A simple Python project that prints 'Hello, World!' to the console.

## Usage
Run the script using Python 3:

```bash
python hello_world.py
```

Or make it executable and run directly:

```bash
chmod +x hello_world.py
./hello_world.py
```

## Requirements
- Python 3.x

## Output
The program will print:
```
Hello, World!
```
