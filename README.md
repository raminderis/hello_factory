# Hello World Project

## Description

A simple Python project that prints "Hello, World!" to the console.

## Usage

To run the program, execute the following command:

```bash
python main.py
```

Or make the file executable and run it directly:

```bash
chmod +x main.py
./main.py
```

## Requirements

- Python 3.x

## Output

The program will output:

```
Hello, World!
```
